你好，我是 **Tong**，目前在跨境电商行业做数据分析。

之前在金融行业待过几年，做过量化策略回测、风险指标监控这些。后来转到跨境电商，主要关注销售数据、用户行为、广告投放效果这块。

平时用 Python 比较多，pandas、numpy 是老朋友了。偶尔也会用 SQL 查数据，Excel 做点快速分析。

#### 工作方向

**金融数据分析**（曾经）
- 量化策略回测框架搭建
- 风险指标计算与监控（最大回撤、夏普比率等）
- 金融时间序列数据处理

**跨境电商数据分析**（现在）
- 销售数据分析与预测
- 用户行为分析
- 广告投放效果评估
- 库存优化建议

**常用工具**
- Python (pandas, numpy, matplotlib, plotly)
- SQL (MySQL, PostgreSQL)
- Excel / Google Sheets
